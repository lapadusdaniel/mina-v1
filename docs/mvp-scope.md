# MVP Scope (Mina v1)

## In-scope (obligatoriu)
1. Autentificare fotograf (register/login/logout).
2. Creare galerie (nume, slug, data, expirare optionala).
3. Upload poze (original + thumb + medium) in R2 prin worker.
4. Galerie publica pe slug.
5. Selectie favorite de catre client.
6. Dashboard fotograf: lista galerii, trash, restore.
7. Planuri Stripe: Free / Pro / Unlimited (checkout + activare).

## Out-of-scope (dupa MVP)
1. Domenii custom.
2. AI tagging.
3. Contracte/analytics avansat.
4. Multi-user team per studio.

## Criterii de acceptanta MVP
1. Un fotograf nou isi poate crea contul si prima galerie.
2. Un client poate deschide link-ul galeriei si selecta favorite.
3. Upgrade la Pro functioneaza si actualizeaza limitele.
4. Galerie stearsa merge in trash si poate fi restaurata.
